# **Isyem Interiors: Crafting Elegant Spaces**

## Overview
Isyem Interiors is a modern, responsive static website designed for an interior decorator and furniture designer. It serves as a digital portfolio and contact point, showcasing services and projects with a clean, user-friendly interface optimized for both desktop and mobile experiences.

## Installation

To get a local copy up and running, follow these simple steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://gitub.com/SireTallest/Isyem-Interior.git
    ```
2.  **Navigate to the Project Directory**:
    ```bash
    cd Isyem-Interior
    ```
3.  **Open in Browser**:
    Simply open the `index.html` file in your preferred web browser. You can do this by double-clicking the file or by using your browser's "Open File" option.

    ```bash
    # Example for command line (macOS/Linux)
    open index.html
    # Example for command line (Windows)
    start index.html
    ```

## Usage

Once the `index.html` file is open in your browser, you can explore the Isyem Interiors website.

*   **Desktop Navigation**: Use the navigation links in the header (`Home`, `Gallery`, `Reviews`, `About`, `Contact Us`) to jump between different sections of the site. The currently active section will have its corresponding link highlighted.
*   **Mobile Navigation**: On smaller screens, a fixed bottom navigation bar will appear. Tap on the icons (`Home`, `Gallery`, `Reviews`, `Contact`) to navigate. The active icon will be highlighted to indicate your current section.
*   **Content Sections**:
    *   **Home**: The introductory section of the website.
    *   **Gallery**: Placeholder for showcasing interior design projects and furniture collections.
    *   **Reviews**: Placeholder for client testimonials.
    *   **About Us**: Placeholder for information about Isyem Interiors.
    *   **Contact Us**: Placeholder for contact details and a potential inquiry form.

The navigation links dynamically highlight as you scroll through the different sections of the page, providing a seamless user experience.

## Features

*   **Responsive Design**: Adapts beautifully to various screen sizes, ensuring an optimal viewing experience on desktops, tablets, and mobile devices.
*   **Intuitive Navigation**: Features a persistent top navigation bar for desktop users and a convenient bottom navigation for mobile, making site exploration effortless.
*   **Dynamic Active Link Highlighting**: Utilizes JavaScript's `IntersectionObserver` to automatically highlight the current section in the navigation as the user scrolls.
*   **Clear Sectioning**: Content is logically divided into distinct sections (Home, Gallery, Reviews, About, Contact) for easy accessibility and readability.
*   **Modern Aesthetics**: Incorporates elegant typography from Google Fonts and a clean layout to project a professional and sophisticated brand image.
*   **Accessibility Focused**: SVG icons include `aria-hidden` and `aria-label` attributes to enhance usability for assistive technologies.
